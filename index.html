Sentence Embeddings for Question Similarity [link]

Project Description

The goal of this project is to build a Quora question similarity classifier that detects duplicate questions. The purpose is to enhance the user experience on Quora by providing better question suggestions, incorporating functionalities such as grammar check and question auto-complete.

Motivation

The motivation behind this project is to enhance the user experience on Quora. Duplicate questions are a common occurrence, and detecting them accurately can help in reducing redundancy and providing more relevant and diverse content to users. By implementing a question similarity classifier, Quora can offer better suggestions, reduce repetitive questions, and enhance the overall quality of the platform.

Dataset

The project utilizes the Quora Question Pairs dataset, which can be accessed at the following link: Quora Question Pairs Dataset. This dataset consists of question pairs, with labels indicating whether the questions are duplicate or not. It serves as the primary resource for training and evaluating the question similarity classifier.

Model Performance

  • Logistic Regression achieved an accuracy of 74.9%, making it the lowest-performing model in the table.
  • The SVM (Support Vector Machine) model achieved an accuracy of 72.45%, which is slightly lower than Logistic Regression.
  • Random Forest outperformed both Logistic Regression and SVM with an accuracy of 76%.
  • LSTM (Long Short-Term Memory), a type of neural network commonly used for sequential data, achieved an accuracy of 80%, showing significant improvement over the previous models.
  • Sentence BERT (Bidirectional Encoder Representations from Transformers) performed the best among all the models with an impressive accuracy of 85%, making it the most accurate model in the table.
