Sentence Bert

BERT has the ability to embed the meaning of words into densely packed vectors and these vectors can be compared using the cosine similarity function of scikit learns. The cosine similarity score would indicate how much similar the 2 questions are.

Steps Followed to get the good Accuracy :
  • Converting the train data columns into an almost matching format with Input Example class of the sentence transformers library.
  • Initializing the data loader.
  • Initializing the model by using bert and pooler modules.
  • Fit MNR loss with the model.
  • Fine tuning our initialized model with the data loader and MNR loss for 1 epoch and saving the model.
  • Calculating the similarity score for a given question pair by using cosine similarity with the trained model.

    • Accuracy: 84.781 %
