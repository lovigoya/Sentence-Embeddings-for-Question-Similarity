LSTM

It uses a convolution network to take account of the local context of words and an LSTM to consider the global context of sentences. This combination of networks helps to preserve the relevant information of sentences and improves the calculation of the similarity between sentences.

EDA and Data Engineering Steps followed to get the better Accuracy for the model:
  • Tokenization the text data using BertTokenizer.
  • Segregating the text data into tokens for question1 and question2 columns with train and test split separately.
  • Model is built by using 3 layers which consists of LSTM and 2 fully connected layers.
  • Setting the values of hyperparameters like num of epochs, learning rate, size of the input, Num of neurons in the hidden layer, num of layers.
  • The loss function is Binary Classification Entropy Loss and optimizer is Adam.
  • Accuracy is calculated by F1 score metric.

    • Accuracy: 64.66 %
