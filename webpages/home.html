Question pairs Classification



Description of the problem:

Building a question similarity classifier to detect duplicate questions in order to provide a better user experience along with incorporating some functionalities like speech recognition and question auto-complete.

Problem Motivation

  • Similar sort of questions are frequently asked in community question answering forums like quora.
  • A lot of time gets wasted for both answer seekers and the ones who answer the questions.
  • The aim is to build a question similarity classifier to detect duplicate questions by calculating the similarity scores using various machine learning algorithms like random forests, logistic regression ,SVM and deep learning architectures like Neural networks, Sentence BERT and LSTM to improve user experience.

    • Data Set Description

    The Quora Question Pairs dataset consists of 404,290 question pairs, provided by Quora

    DataFields:
  • id(int) - the id of a training set question pair.
  • qid1(int), qid2(int) - unique ids of each question (only available in train.csv)
  • question1(object), question2(object) - the full text of each question
  • is_duplicate(int) - the target variable, set to 1 if question1 and question2 have essentially the same meaning, and 0 otherwise.
